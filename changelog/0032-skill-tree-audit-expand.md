# 0032 · 技能樹全面稽核 + 節點數量倍增

- 日期：2026-05-30

## 摘要
逐一稽核每一棵技能樹的每個節點，確認沒有「死碼／無效加成」，
並把全部技能樹的節點數量大約翻倍，讓深度發展更有層次。

## 稽核（先確認沒有沒用或不合理的節點）
- 寫了一支驗證腳本檢查全部 7 棵樹：**無重複 id、所有 parent 都存在、每棵樹剛好一個根**。
- 比對 `engine/stats.js` 實際消耗的 key，確認**每個 bonus key 都會被該武器吃到**：
  - 雷射／火焰為「持續型」武器，正確地**不含** `rateM`、`multishot`(火焰)、`splash` 等用不到的加成。
  - 修掉了之前不小心混進基礎樹的**重複 `b_pen` 節點**。
  - `nodeDesc()` 補上先前沒有文案的 key（荊棘／無人機／吸血／分裂／減速／灼燒頻率／聚焦增幅／碎片），所有節點說明都能正確顯示。

## 擴大（節點數量約翻倍）
- 基礎屬性樹：19 → **34**（攻擊／防禦／經濟三主幹各往下延伸，新增荊棘、無人機群、吸血、鷹眼等支線）。
- 各武器樹（只用該武器有效的 key）：
  - 標準彈 11 → **19**（傷害暴擊／連射彈幕／穿透濺射 + 破甲支線，新增第二招牌「鋪天蓋地」）。
  - 追蹤彈 10 → **18**（新增破甲支線與「蜂群」招牌）。
  - 雷射 11 → **19**（多束分光線拉長、聚焦滾雪球線加深、獨立熔甲支線）。
  - 折射 11 → **19**（彈射分裂線延伸到「無盡迴路／鏈式裂變」、破甲支線）。
  - 火焰 10 → **19**（火域擴張／減速控場各加深，獨立熔甲支線與「絕對零度」招牌）。
  - 分裂彈 8 → **16**（碎片數線、速射多重線、穿甲支線都補滿）。
- 武器樹合計 61 → **110**；全遊戲技能節點合計 **144**。

## 測試
- 驗證腳本：重複 id 0、壞 parent 0、無效 key 0、根節點數正確。
- `npm run build` 通過。
- headless 模擬器：首玩仍約 28 波；全永久投入由約 38 波提升到 **41–53 波**，永久進度的深度更有回報，且未失衡。

## 影響範圍
- `data/skillTree.js`（節點資料 + 文案）。不改存檔格式：既有玩家的已點節點沿用，新節點直接可點。
