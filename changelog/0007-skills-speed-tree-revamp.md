# 0007 · 技能調整 + 倍速 + 敵人成長上限 + 直排技能樹擴大 + 存檔遷移

- 日期：2026-05-29

## 摘要
局內技能新增彈速/濺射範圍、移除多重；子彈基礎速度 −75%；新增遊戲內倍速；敵人數量/移速改為微幅成長且有上限；技能樹改為直排樹狀並擴大、加更多基礎數值節點；自動存檔改為 JSON 並從舊格式無痛遷移。

## 細項
- **局內技能**（`data/skills.js`）：攻擊新增「彈速」、特殊新增「濺射範圍」、移除攻擊「多重」（多重改由技能地圖節點提供）。
- **子彈速度**：`WORLD.bulletSpd` 4.2 → 1.05（−75%）；`derive` 新增 `bulletSpeed`、`splashRadius`；`update.js` 改用之；子彈存活時間拉長至 3.4s 以配合慢速。
- **倍速**：`App` 新增 1×/2×/4×（迴圈以多子步驟模擬）；按鈕在戰場左上角。
- **敵人成長**（`data/tuning.js`）：`countSlope` 1.0→0.6 且 `countCap=22`；`spdScaleBase` 1.015→1.012 且 `spdScaleCap=1.9`。
- **技能地圖**：改為直排樹狀（核心在上、依深度往下生長），並新增 9 個基礎數值節點（火力/連射/碾壓/裝甲/活力/神盾/財源/精算/滿溢），共 53 節點；`SkillMap` 預設視角調整以框住較高的樹。
- **存檔遷移**：本機自動存檔改為 JSON（`thetower_meta`），對新增節點前向相容、不再被清空；啟動時自動從舊版 v3 字串（`legacyDecodeV3`）遷移鑽石/節點/最佳波次。進度代碼版本提升為 `4`。
- 數值面板新增「子彈速度」「濺射範圍」列。

## 影響範圍
- 進度代碼格式版本 `3`→`4`（舊「分享代碼」失效；但本機進度會自動遷移保留）。
