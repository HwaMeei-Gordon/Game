# 0030 · UI 人類視角檢查：精簡 HUD + 暫停選單整合

- 日期：2026-05-29

## 問題（人類視角）
- 遊戲內頂部一排塞了 8 個東西（返回/波次/金幣/鑽石/難度/📊/👾/暫停），手機上**金幣鑽石數字會被擠到截斷**，且 📊👾 emoji 不直覺。

## 調整
- **HUD 精簡**：只留即時資訊（返回 · 波次或時間/擊殺 · 金幣 · 鑽石 · 暫停），數字有更多空間不再截斷；血條加高更好讀。
- **暫停選單整合**：把「數值面板 / 敵人圖鑑 / 設定 / 主選單」收進暫停選單（附文字標籤、加大按鈕），更清楚好按；開面板仍會自動暫停。
- 移除 HUD 上低價值的難度膠囊（開局與結算都看得到）。

## 影響範圍
- 純 UI/UX；不影響數值或存檔。
