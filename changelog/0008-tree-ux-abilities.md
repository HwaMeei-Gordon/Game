# 0008 · 技能樹防誤觸/放大/花費預覽 + 超級範圍 + 主動技能說明

- 日期：2026-05-29

## 摘要
改善技能地圖的操作與可讀性，新增可突破射程上限的防禦節點，並讓遊戲內主動技能看得懂。

## 細項
- **防誤觸**：技能地圖「點一下只選取」，購買/取消一律走下方按鈕（二次確認），不再點到就花鑽石。
- **超級放大**：版面整體放大（`SPREAD = 1.7`、viewBox 2400、節點略放大、預設縮放 0.6），節點不再擁擠；縮放下限放寬到 0.4。
- **花費預覽**：每個未點亮節點下方直接顯示 `💎花費`（可負擔=亮色、不足=灰色、未解鎖加 🔒）。
- **防禦新增「鷹眼 → 超級範圍」**：以 `rangeFlat` 提供攻擊範圍加成，可突破原本射程上限（`rangeOf` 加上 `s.rangeFlat`）；建議在局內「範圍」點滿後取得。
- **主動技能說明**：過載/新星/冰霜/修復按鈕可**長按看說明**（效果、冷卻、持續時間）；並在技能列上方加提示文字。

## 影響範圍
- 技能地圖節點數 53→55；本機 JSON 存檔前向相容、進度不受影響。
