# 0017 · 結算畫面（波次/擊殺/時間/獲得鑽石 + 各武器傷害佔比）

- 日期：2026-05-29

## 摘要
回合結束顯示結算：擊殺、存活時間、本局獲得鑽石，以及各武器的傷害佔比長條，讓 build 多樣性可視化、給目標感。

## 細項
- `game.js`：新增 `tagDmg` 累計各武器傷害；`g.wdmg`、`g.runGems`。
- `update.js`：雷射/火焰/子彈/折射命中時累計傷害；過波與結算時累計本局鑽石。
- `App.jsx`：遊戲結束時組出 summary（含各武器傷害佔比），傳給結算畫面；開新局清除。
- `GameScreen.jsx`：結算畫面顯示數據與各武器傷害佔比長條。

## 設計參考
- 參考塔防 roguelite 設計：build 多樣性（economy/utility/offense）與「回合結束回饋」。

## 影響範圍
- 純呈現/統計；不影響數值或存檔。
