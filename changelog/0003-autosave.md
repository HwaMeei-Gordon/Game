# 0003 · 本機自動存檔（localStorage）

- 日期：2026-05-29

## 摘要
進度（鑽石、技能地圖、最佳波次）自動保存在玩家裝置的瀏覽器，關閉/重整後仍保留；進度代碼改為「換裝置/備份」用途。

## 細項
- `App.jsx`：以 `SAVE_KEY = thetower_save_v3` 在啟動時從 localStorage 讀取進度（沿用 `decodeSave` 驗證），
  並用 `useEffect([metaV])` 在每次進度變動時 `encodeSave` 寫回；try/catch 包覆（無痕模式不可用時不報錯）。
- 主選單說明文字更新為「自動存在本機，換手機請用進度代碼」。

## 影響範圍
- 不改變存檔格式（仍為版本 `3`），代碼與本機存檔互通。
- GitHub Pages 僅託管程式，不保存個人進度；進度只在玩家本機。
