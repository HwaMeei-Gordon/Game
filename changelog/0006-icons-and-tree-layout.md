# 0006 · 全新手繪圖示 + 技能樹有機佈局

- 日期：2026-05-29

## 摘要
重畫所有圖示為有立體質感的手繪 SVG（漸層＋高光＋描邊），並把技能地圖改成有機散佈、不規則但仍散開的流動佈局。

## 細項
- `components/Icon.jsx`：全部圖示重畫。每個圖示用「亮面→主色→暗面」垂直漸層填色、加上白色高光與深色描邊做出立體感；漸層 id 以 `useId` 確保唯一不衝突；顏色由 `color` 衍生，沿用各處主題色。
  - 重新設計：寶劍(攻擊)、閃電(攻速)、雷達靶心(範圍)、三發箭(多重)、穿透箭(穿甲)、心(生命)、十字(恢復)、盾(護甲)、四角星(暴擊)、爆裂星芒(濺射)、尖刺星(荊棘)、錢幣(金幣)、寶石(鑽石)、環繞球(無人機)、反應爐(核心)、骷髏(詛咒)、砲塔(標準彈)、火箭(追蹤彈)、發射器光束(雷射)、彈射閃電(折射)、火焰。
- `data/skillTree.js`：節點座標改為有機散佈（三條藤蔓自核心蜿蜒而出，帶弧度與抖動、刻意不對齊），仍保持彼此散開、連線清楚。拓撲（parent/連結）不變。

## 影響範圍
- 純視覺調整，不影響數值、存檔格式或既有進度。
